# Monopoly-Land-Frequency
This program shows you the chance that a player will land on any particular tile in Monopoly

This program is able to show you the number of times a player has landed on the tile, the ratio compared to the average tile, or the percent of landed on a specific tile.

The program also shows analytical data on how much rent a property gives, how much money you need to spend to get it to that level of rent, the chance of landing on particular set, as well as analytical datat shoing the average land rate for all sets.

This program is meant to show the relative worth of each property in monopoly

This program also accounts for the mechanics of "Go to Jail", "Chance", "Community Chest", "Rolling Doubles" etc that affect where the player lands on the board.

This program uses the British version of monopoly. Which matters specifically in how many Chance and Community Chest cards are able to move the players.

By default the program counts every times the player lands on a square so that even though the player imediately leaves the "Go To Jail" tile it still counts it visiting there. Though this can be changed.
